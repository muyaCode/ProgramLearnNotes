

# 密探工具集GUI渗透版

## 01 介绍

密探-主要包含域名信息查询，IP端口查询，备案信息查询，搜索引擎语法自动生成（FOFA、Hunter、google,github），文件扫描（包含目录，备份文件，spring信息泄漏，自定义字典等）、渗透技能路线备忘录，常用网络安全网站导航等功能。

## 02 安装与使用

在jdk8环境下运行以下语句运行:

```
java -jar mitan-1.0-jar-with-dependencies.jar
```
